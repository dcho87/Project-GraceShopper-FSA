### *A Fullstack e-commerce website featuring product browsing and purchasing NFTs*
---
## Main Technologies
---
Javascript, React, Redux, Express, Node.js, Sequelize, PostgreSQL, Passport, OAuth, Mocha, Chai

## Key Features
---
- Authorize admin access to add/remove/edit product and user information
- Create an account to view previous purchase history
- Navigate to view all products and single product page for more detail
- Product with same category on single product page
- Purchase items as a guest user
- Add/remove/edit order cart

## To Use
---
1. Fork and clone the repo
2. npm install
3. Create postgres db "graceshopperteam5"
4. npm run seed
5. npm run start
